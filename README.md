# 🎮 Juego Fake or Real - Android Studio  

## 📖 Descripción  
Juego interactivo desarrollado en Kotlin con Jetpack Compose que reta a los jugadores a identificar si una imagen fue creada por IA o por un humano.

## 🛠️ Funcionalidades  
- Presentación aleatoria de imágenes generadas por IA y reales.  
- Sistema de puntuación basado en respuestas correctas.  
- Animaciones dinámicas y efectos visuales con Jetpack Compose.  
- Base de datos para almacenar puntuaciones de los jugadores.  

## 🚀 Tecnologías utilizadas  
- Android Studio  
- Kotlin  
- Jetpack Compose  
- Firebase para almacenamiento de datos  

## ▶️ Cómo ejecutar el proyecto  
1. Abrir en Android Studio.  
2. Instalar dependencias requeridas.  
3. Compilar y ejecutar en un emulador o dispositivo real.  
4. Iniciar sesión con Google para guardar el progreso.
